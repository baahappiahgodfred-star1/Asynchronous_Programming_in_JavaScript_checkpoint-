# Async Programming in JavaScript Checkpoint

## Project Overview

This project demonstrates the use of asynchronous programming in JavaScript using:

- async / await
- Promise
- Promise.all()
- Error handling with try/catch

The project contains solutions for multiple asynchronous programming tasks.

---

## Tasks Implemented

### Task 01
Iterating with Async/Await using delayed logs.

### Task 02
Simulating an API call using async/await.

### Task 03
Handling API errors using try/catch.

### Task 04
Making concurrent requests using Promise.all().
